301 Redirect Manager

How to use this tool:

  1. Step 1: Upload old URLs CSV (from Screaming Frog) or manually add old URLs
  2. Step 2: Upload new URLs CSV or manually add new slugs
  3. Step 3: Enter target URLs for unmatched URLs that need mapping (highlighted at the top with red borders)
  4. Step 4: Verify all mappings, then click "Export to CSV" to download a redirects.csv file

Note: URLs with status codes 301 or 404 are automatically filtered out from the old URLs.

Important: If your site already has existing redirects in place, you'll need to manually add these as old URLs to ensure they're included in your final redirect list.

Pro Tips:

  • You can add old URLs individually using the manual entry option - useful for small sets of URLs
  • URLs needing attention are at the top with empty target fields - add a mapping to turn them green
  • The status will change to "Mapped" once you enter a target URL
  • URLs with exact matches are moved to the bottom and grayed out
  • You can set a URL's action to "Skip" from the dropdown (it will be crossed out)
  • Use the bulk action buttons to quickly manage multiple redirects at once

Domain Settings (Optional)

If your URLs are already in slug format (e.g., "/path/to/page"), you can leave this blank.
